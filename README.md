# myhouseonthecorner.com
Docker compose project for myhouseonthecorner.com. Ghost blog, fgallery, with docker volumes.

### Build instructions.

`make build;`

### Run instructions.

`make run;`

### Stop all containers.

`make stop;`

### Remove the built containers.

`make clean;`

### Remove dangling images.

`make clean-images;`
